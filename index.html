Embedded Machine Learning
This repository contains a variety of projects and exercises exploring machine learning on embedded systems. Highlights include training convolutional neural networks (CNNs), model quantization, data aggregation workflows, and hands-on experiments. Notable projects include creating a DIY "voice assistant" using ESP32-S3, face recognition with a tiny camera, and optimizing models for real-time performance on resource-constrained devices.

The final project, featured in the video showcase, focuses on Autonomous Vehicle Racing. Using ESP32-S3 alongside RPLidar A1, we demonstrated how a tiny microcontroller can power and control an F1TENTH car. This project highlights the potential of embedded systems in driving a scaled RC car autonomously.

Facial Emotion Recognition Project

This project explores the development and evaluation of a Facial Emotion Recognition system using deep learning models. The study applies models such as AlexNet, CNN, ResNet, DenseNet, and DeiT on a dataset of over 28,000 images to classify emotions into seven categories. The analysis highlights the benefits, limitations, and comparative performance of these models, with AlexNet emerging as the top-performing approach. The project addresses challenges like data imbalance and hardware limitations while proposing future improvements for better generalization and accuracy. GitHub Repository

Impact of Public Library Resources on K-12 Education in Kansas

This project investigates the potential impact of public library resources on educational outcomes in Kansas public schools. Using data from the Public Libraries Survey, U.S. Census Bureau, and educational test results like ACT and RLA scores, the analysis explores correlations, regression models, and classification methods. Despite weak results, patterns emerged that suggest further study could guide resource allocation to improve educational outcomes. The project highlights the complexity of factors influencing education and provides insights for long-term analyses. GitHub Repository
